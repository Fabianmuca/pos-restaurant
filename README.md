# 🍷 BarPOS — Sistem POS për Bar-Restorant

Sistem i plotë i menaxhimit të pikës së shitjes (Point of Sale) për bar-restorantet.

## 🚀 Live Demo
- **Frontend:** https://pos-restaurant-zeta.vercel.app/
- **Backend:** https://pos-restaurant-backend-c9ow.onrender.com

## ✨ Funksionalitetet
- 🪑 Menaxhim tavolinash me auto-refresh
- 📋 Meny digjitale me CRUD të plotë
- 👨‍🍳 Panel kuzhine në kohë reale
- 💳 Pagesa me cash dhe kartë
- 🖨️ Printim faturash POS 80mm
- 🔖 Rezervim tavolinash
- 🔐 Role: Admin, Kamarier, Arketar

## 🛠️ Teknologjia
- **Frontend:** React, Redux Toolkit, RTK Query
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT + bcryptjs
